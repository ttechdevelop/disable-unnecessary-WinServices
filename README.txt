// Инструкция по использованию

- Для запуска скрипта используйте PowerShell, запустите от имени администратора

- Введите команду для перехода к директории с сохраненным файлом. Пример: cd C:\Users\ВашеИмяПользователя\Desktop

- Введите команду для запуска скрипта: .\DisableUnnecessaryWinServices.ps1



Возможые ошибки:

1)Скрипты отключены на этом компьютере

При ошибки с выводом сообщения "Скрипты отключены на этом компьютере" требуется проверить политику ExecutionPolicy, для этого введите команду Get-ExecutionPolicy. 

Убедитесь, что политика установлена, как RemoteSigned или Unrestricted. 

Если политика установлена в режим Restricted, требуется ввести команду: Set-ExecutionPolicy Unrestricted или Set-ExecutionPolicy RemoteSigned или запросить доступ у локального администратора для предоставления доступа.
